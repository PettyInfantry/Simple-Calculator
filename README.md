# Simple Python Calculator

A clean, interactive command-line calculator written in Python.

## Features
- Basic arithmetic operations: addition (+), subtraction (−), multiplication (×), division (÷)
- Clear prompts and formatted result output
- Input validation for numbers
- Protection against division by zero
- Press `q` at any input prompt to quit immediately
- Option to perform multiple calculations in one session

## Requirements
- Python 3.6 or higher  
(No external libraries required)

## Installation
1. Clone or download this repository
2. Open a terminal/command prompt in the project folder

## Usage
Run the program:

```bash
python calculator.py
